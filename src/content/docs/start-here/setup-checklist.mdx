Most new media library deployments in LightNet follow a similar setup process. 
To make setup easier and more consistent, we maintain a checklist of common tasks 
that are typically completed during a new installation.

You can use this checklist for your own deployment. 
We recommend copying it into a GitHub issue so that tasks can be tracked and completed as you work through the process.

Not every item will apply to every ministry or organization. 
If a task is not relevant, either remove it or mark it as completed by striking it through using Markdown syntax (~~text~~).
