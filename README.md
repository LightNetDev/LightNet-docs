# LightNet developer docs

This repository contains the LightNet developer documentation, which you can also [view online](https://docs.lightnet.community).

## 📦 Project commands

All commands are run from the root of the project, from a terminal:

| Command          | Description                                                   |
| ---------------- | ------------------------------------------------------------- |
| `pnpm install`   | Install all project dependencies                              |
| `pnpm dev`       | Start the local development server at `localhost:4321`        |
| `pnpm build`     | Run checks and build the production site to the `./dist/` folder |
| `pnpm preview`   | Preview the production build locally                          |
| `pnpm linkcheck` | Validate internal links, then run checks and build the site   |
